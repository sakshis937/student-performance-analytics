# 📊 Student Performance Analytics

A data analytics dashboard built using **Python, SQLite, Pandas, Matplotlib, and Streamlit** to analyze student performance, identify at-risk students, and generate educational insights.

---

## 🚀 Features

### 📈 Dashboard Analytics

* Total Students Overview
* Average Grade Analysis
* Highest & Lowest Grade Metrics
* Attendance Analysis
* Study Time vs Performance

### 🏆 Student Performance

* Top 10 Performing Students
* Gender-wise Performance Analysis
* School-wise Performance Comparison
* Parent Education Impact Analysis

### 🌐 Behavioral Insights

* Internet Access vs Grades
* Weekend Alcohol Consumption Impact
* Weekday Alcohol Consumption Impact
* Failure History Analysis

### ⚠️ At-Risk Student Detection

Students are flagged as at-risk if:

* Final Grade (G3) < 10
* Previous Failures > 0
* Absences > 10

### 📥 Data Export

* Download filtered student reports as CSV

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sakshis937/student-performance-analytics.git
cd student-performance-analytics
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Streamlit App

```bash
streamlit run app.py
```
